Control AngularJS Only Numbers para PISCYS

Objetivo:Permitir la entrada de solo números decimales,enteros o valores de moneda.

Desarrollado con los mismos criterios de los campos en AS DIRECT con HTML, CSS, AngularJS y Javascript.

Ejemplo básico:
Código: Javascript de Vista dentro del controlador
    $scope.Datos = {
        MontoOne: 1504.00,
        ... 
    }
Código: HTML de Vista
                                                    
1<control-only-numbers id="exampleOne" ng-model="Datos.MontoOne" label="Básico">
2</control-only-numbers>

Las propiedades obligatorias son model e id. El primero representa la relacion de la vista con el controlador Angular.js, la propiedad ng-model. La segunda es un id que identifica al control, es importante en caso que se encuentre mas de un control de este tipo en la vista y una mejora con respecto a los controles de AS DIRECT que automaticamente se genera un objeto en el $scope padre con el id provisto por la propiedad.

la propiedad label genera un etiqueta para el control con el string pasado por valor.
